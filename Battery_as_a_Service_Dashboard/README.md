# 🔋 Battery-as-a-Service Performance Dashboard

## 📌 Project Overview
This dashboard analyzes the performance of an EV Battery-as-a-Service (BaaS) network.

## 🛠 Tools Used
- Power BI
- Power Query
- DAX

## 📊 KPIs
- Total Revenue
- Total Swaps
- Total Stations
- Average Swaps
- Station Uptime %

## 📈 Dashboard Features
- Revenue by City
- Revenue by Partner
- Station Uptime
- Battery Health
- Forecasting
- Trend Analysis
